# Instalación de Breeze

## Crear proyecto

    composer create-project laravel/laravel nombre

## Crear base de datos *
> Podríamos no crear la base de datos, ya que la migración nos pregunta si queremos que Laravel la cree

## Configurar .env
> Configurar la sección de base de datos ( nombre )
    
## Correr migraciones

    php artisan migrate  

## Descargamos breeze
  
    composer require larave/breeze --dev

## Instalar breeze

    php artisan breeze:install

> ***Nota:*** Requiere npm instalado
